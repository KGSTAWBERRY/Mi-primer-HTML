Seccion de noticias: como se utilizan las etiquetas

Noticia #1

La etiqueta (h1) es el titulo principal de la pagina web (mi primer pagina web), la etiqueta (h2)coloca un subTitulo (Seccion de noticias) la etiqueta (h3) crea pequeÑos subtitulos como los que estan nombrados como (noticia #1) en esta pagina web, la etiqueta (p) es con la que introduciomos textos debajo de los subtitulos como este parrafo, la etiqueta (lorem) crea texto de relleno como el siguiente:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Pariatur voluptatibus rem alias, corporis aliquam nobis consequuntur cupiditate quos ducimus reprehenderit expedita, cum, impedit autem inventore unde eos vitae obcaecati velit.

Noticia #2

Las etiqutas (ol), con (li) dentro permiten hacer una lista emnumerada automaticamente como la siguiente:

  1. este es el item
  2. este es el item
  3. este es el item

Noticia #3

Con las etiquetas (ul), con (li) hacemos otro listado de pero nin la emnumeracion o para hacer un listado que no vallamos a emnumerar o que no sea necesario emnumerar como el siguiente

  • este es un elemento
  • este en un elemento
  • este es un elemento

Noticia #4

la etiqueta (span) sirve para poner textos cortos de seguido en la misma seccion o linea como la siguiente que esta ubica en el encabezado de la pagina
